# ChineseLearn 國小生字學習App

這是一個專為國小學生設計的中文生字學習網頁應用程式，提供生字認識、注音拆解朗讀以及配對遊戲功能。

## ✨ 功能特色

- 📚 **年級選擇**: 支援國小1-6年級上下學期生字
- 🎧 **智慧朗讀**: 點擊生字自動朗讀，包含注音拆解與聲調說明
- 🎴 **學習模式**: 卡片式瀏覽生字、注音、範例詞語
- 🎮 **配對遊戲**: 漢字與注音配對小遊戲，加強記憶
- 📊 **學習進度**: 自動記錄學習進度與熟練度
- 💾 **本地儲存**: 所有學習記錄儲存於瀏覽器中，不需伺服器

## 🚀 如何使用

1. 直接用瀏覽器打開 index.html 即可使用
2. 選擇年級與學期
3. 在「認識新字」模式閱讀生字
4. 累積足夠字數後可進入「配對遊戲」
5. 點擊漢字即可聽到發音與注音說明

## 🛠️ 技術架構

- 原生 JavaScript (無框架依賴)
- Tailwind CSS 樣式
- Web Speech API 語音合成
- LocalStorage 本地儲存
- 純前端靜態網站，不需要後端伺服器

## 🎯 設計理念

- 簡單直覺的操作介面，適合兒童使用
- 重複間隔學習法，針對不熟的字加強練習
- 注音拆解教學，幫助正確發音
- 遊戲化學習，提升學習動機
